# ViVeTool Manager v3.9

## 项目简介

ViVeTool Manager v3.9 是一款功能强大的 Windows 隐藏功能管理工具，采用风格设计的图形界面。本工具可以帮助用户轻松启用或禁用 Windows 系统的各种隐藏功能，支持中英文界面切换，并提供完整的操作日志和状态反馈。

## 主要功能

- **智能搜索**：自动在系统常见目录中搜索 ViVeTool 文件夹
- **手动浏览**：支持用户手动选择 ViVeTool 所在路径
- **功能管理**：添加、查看、清除和恢复默认功能 ID
- **一键操作**：快速启用或禁用选中的隐藏功能
- **双语支持**：支持中文和英文界面无缝切换
- **日志记录**：完整的操作日志显示，实时反馈执行状态
- **重启提示**：操作成功后提示用户重启计算机以应用更改
- **管理员权限**：自动检测并请求管理员权限

## 系统要求

- 操作系统：Windows 10/11
- Python 版本：Python 3.6 或更高版本
- 依赖模块：tkinter（Python 标准库）
- ViVeTool：需要下载并放置在可访问的目录中

## 使用方法

### 启动程序

直接双击 `run.bat` 文件即可启动程序。如果当前不是管理员权限，程序会自动提示并询问是否以管理员身份重新运行。

### 基本操作流程

1. **设置 ViVeTool 路径**：程序启动后会自动搜索 ViVeTool 文件夹，如果未能自动找到，可以点击「智能搜索」或「浏览文件夹」手动选择
2. **管理功能 ID**：在功能管理区域可以查看当前的功能 ID 列表，支持添加新的功能 ID、清空列表或恢复默认
3. **执行操作**：选择需要启用或禁用的功能，点击对应的按钮进行操作
4. **查看日志**：右侧日志区域会显示详细的执行过程和结果
5. **重启计算机**：操作成功后，点击「立即重启」按钮重启计算机使更改生效

### 语言切换

点击界面右上角的「English」或「中文」按钮，可以在两种语言之间切换。所有界面文本、按钮标签和提示信息都会实时更新。

## 功能说明

### 配置区域

- **智能搜索**：自动在 Downloads、Desktop、Documents 等目录搜索 ViVeTool 文件夹
- **浏览文件夹**：打开系统文件夹选择对话框，手动定位 ViVeTool 路径
- **路径输入框**：显示当前设置的 ViVeTool 路径

### 功能管理区域

- **功能列表**：显示当前选中的所有功能 ID
- **添加功能**：输入功能 ID 后按回车或点击「添加」按钮
- **清空列表**：清除所有选中的功能 ID
- **恢复默认**：将功能 ID 列表恢复为默认值

### 操作按钮

- **启用功能**：以管理员身份执行 ViVeTool 启用命令
- **禁用功能**：以管理员身份执行 ViVeTool 禁用命令
- **清空日志**：清除日志区域的所有内容

### 日志区域

- **执行日志**：显示所有操作记录和系统反馈
- **结果提示**：显示操作成功或失败的详细信息
- **重启按钮**：操作成功后显示，用于快速重启计算机

## 常见问题

### 问：程序提示找不到 ViVeTool？

请确保已将 ViVeTool 下载到本地，然后点击「浏览文件夹」按钮手动选择 ViVeTool 所在目录。常见的下载位置包括 Downloads 文件夹或桌面。

### 问：操作执行失败怎么办？

请检查以下几点：是否以管理员身份运行程序、ViVeTool 路径是否正确、功能 ID 是否有效。如果问题仍然存在，请查看日志区域的错误信息。

### 问：为什么需要重启计算机？

Windows 系统的大多数隐藏功能需要重启后才能生效。程序会在操作成功后自动提示您重启计算机。

### 问：如何添加自定义功能 ID？

在「功能 ID」输入框中输入数字 ID，然后按回车键或点击「添加」按钮即可。添加的功能会立即显示在功能列表中。
